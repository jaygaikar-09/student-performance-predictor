STUDENT PERFORMANCE PREDICTOR

A Machine Learning-based application that predicts a student's exam score (0–100) based on academic and lifestyle factors.

---

OVERVIEW

This project analyzes factors like study hours, attendance, mental health, sleep duration, and part-time job status to predict student performance using a trained ML model.

It provides:

* Command Line Interface (CLI)
* Interactive Web App (Streamlit)

---

FEATURES

* Predicts student exam score (0–100)
* Machine Learning-based prediction
* Interactive Streamlit UI
* CLI-based execution
* Easy and user-friendly interface

---

TECH STACK

* Python
* NumPy
* Pandas
* Scikit-learn
* Streamlit

---

INPUT PARAMETERS

* Study Hours per Day
* Attendance Percentage
* Mental Health Rating (1–10)
* Sleep Hours per Night
* Part-Time Job (Yes/No)

---

HOW IT WORKS

1. User inputs data (via CLI or UI)
2. Data is preprocessed (encoding, formatting)
3. Input is passed to trained ML model (best_model.pkl)
4. Model predicts exam score
5. Output is displayed to user

---

HOW TO RUN

Run CLI Version:
python main.py

Run Web App:
streamlit run app.py
